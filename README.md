\# MW-BOT (Production Base)



MW-BOT is the official Mageworld Discord bot.



\## Current Features

\- Slash commands (/ping, /status)

\- Railway / VPS ready

\- Safe structure for future upgrades



\## Deploy (Railway)

1\. Push to GitHub

2\. Railway → New Project → Deploy from GitHub

3\. Add DISCORD\_TOKEN in Variables

4\. Done
